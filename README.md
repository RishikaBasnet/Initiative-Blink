# Initiative-Blink
![Initiative Blink](Assets/Images/BlinkLogo.png)

## Make memories, Not Decisions... in the blink of an eye...

Initiative Blink was created as a hackathon project for **IslandHacks 2025**, my first experience building a project of this scale with minimal prior experience in web development. The same also applies to the lovely teammates who assisted me in making this website come together. The project was developed as an opportunity to learn by building, allowing me to explore web design, HTML, CSS, and JavaScript while creating a platform centered around event planning for people of various needs. This cute website allows you to take mini quizzes and then tells you the activitiy you should do based on your group size. Despite my limited experience at the time, the project went on to win **Best Beginner Project** at IslandHacks 2025. This repository preserves the development and structure of the website as an early milestone in my journey as a developer.

## Repository Overview

This repository contains the source files for Initiative Blink. 

The repository is primarily divided into two main folders:

* **`assets/`** — Contains the website's visual and design assets, organized into several subfolders:

  * **`favicon/`** — Contains the website favicon files
  * **`images/`** — Contains images used throughout the website
  * **`headers/`** — Contains header images 
  * **`buttons/`** — Contains button graphics

* **`pages/`** — Contains the files for each individual page of the website. Each page is organized into its own folder to keep its associated files together.

## Shared Files

The website also contains files that apply across multiple pages:

* **`styles.css`** — Contains the shared CSS styling used throughout the website. This file controls the overall visual appearance, layout, typography, and other common design elements across the pages.

Using a shared stylesheet allows the website to maintain a consistent visual design across its different pages while avoiding unnecessary duplication of CSS code.

## Viewing the Full Website

To view the full Initiative Blink website, you can host the website using **GitHub Pages**:

1. **Fork this repository** to create your own copy.
2. Open your newly created fork and navigate to the **Settings** tab.
3. Under the **Code and automation** section in the sidebar, select **Pages**.
4. Under the **Source** section, select the **`main`** or **`master`** branch as the source.
5. Click **Save**.
6. GitHub will generate a live **`github.io`** URL for your fork. The website may take a few minutes to become available.

Once the site has been deployed, you can use the generated `github.io` URL to view the full website in your browser.
